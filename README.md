# Lazzy Badger Portfolio 🦡

A high-performance, cinematic technical portfolio built for **Anudit Khatri** (aka Lazzy Badger). This site showcases expertise in software development, cybersecurity research, and AI systems orchestration.

## 🚀 Live Demo
[https://anuditk.vercel.app](https://anuditk.vercel.app)

## 🛠 Tech Stack
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Data Management**: Local JSON-based profile system (`src/data/profile.json`)

## ✨ Key Features
- **Cinematic UI**: Dark-themed, hacker-inspired aesthetic with gold and cream accents.
- **Ambient Stage**: Dynamic background effects for an immersive experience.
- **Scroll Orchestration**: Smooth, revealed-on-scroll animations and transitions.
- **Responsive Design**: Optimized for everything from mobile devices to ultrawide monitors.
- **System Showcase**: Dedicated sections for projects, skills, and a visual gallery of field notes.

## 📦 Installation & Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/arthurrgg8-lgtm/lazzy-badger-portfolio.git
   cd lazzy-badger-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the result.

## ⚙️ Configuration
The website content is driven by `src/data/profile.json`. To update the portfolio:
1. Modify `src/data/profile.json` with new identity, projects, or social links.
2. The UI will automatically reflect these changes upon rebuild.
